# 🏥 Hospital Management System (HMS)

A full-stack web application built using Flask to streamline hospital operations including patient management, doctor coordination, appointment scheduling, and administrative control.

> 🚀 Designed to simulate real-world hospital workflows with role-based dashboards and scalable architecture.

---

## 🌟 Key Features

### 🔐 Authentication & Roles

* Secure login system for **Admin, Doctor, and Patient**
* Role-based access control

### 👨‍💼 Admin Dashboard

* Manage doctors (Add / Edit / Delete)
* Manage departments
* View all registered patients

### 👨‍⚕️ Doctor Dashboard

* View assigned patients
* Update patient medical history
* Manage availability schedule

### 🧑‍🤝‍🧑 Patient Dashboard

* Register & manage profile
* Book appointments
* View medical history
* Check doctor availability

## ⚙️ Setup & Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Amandeep-winner/hospital-management.git
cd hospital-management
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
python app.py
```

🌐 App runs at: http://127.0.0.1:5000/
